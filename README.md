# 📅 2026年4月 考勤日历

一个交互式的考勤日历应用，支持加班/休息时长管理，集成实时天气显示。

## ✨ 功能特点

- **考勤管理**：点击日期快速切换加班/休息状态
- **自定义时长**：支持对每一天单独设置加班获得/休息花费时长
- **实时天气**：
  - 显示过去7天 + 未来16天天气预报
  - 悬浮日期格子显示详细天气信息
  - 支持自定义城市查询
- **数据持久化**：所有数据保存在浏览器本地，刷新不丢失
- **今日提醒**：当前日期特殊高亮标注

## 🚀 在线使用

直接访问：`https://你的用户名.github.io/calendar-202604/`

## 📦 本地使用

1. 下载 `index.html` 文件
2. 双击用浏览器打开即可使用

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript，无需任何框架
- 天气数据：Open-Meteo API（免费、无需API密钥）
- 数据存储：浏览器 localStorage

## 📝 使用说明

1. **设置默认时长**：在"默认时长设置"中配置加班获得和休息花费的默认小时数
2. **标记日期**：点击日历格子循环切换状态（普通 → 加班 → 休息 → 普通）
3. **自定义时长**：悬浮日期后点击✏️按钮，可单独设置该天的时长
4. **查看天气**：
   - 悬浮日期格子查看简要天气
   - 点击日期或时间轴查看详细天气
   - 输入城市名查询其他地区天气
